    Adding new Questions to Cozie basic

    Cozie determines the questions to be asked using "flows", json files in the resources/flows directory.

    To define a new Flow, simply create a new json file in this directory and point the questionsFlow variable in app/Index.js to it.

    Flow files

    The basic layout of a flow file is as follows:

    export default [
    {
        name: "thermal",
        type: "icon",
        requiresAnswer: [],
        questionText: "Would you prefer to be?",
        questionSecondText: "",
        iconText: ["Cooler", "Warmer", "No change"],
        iconColors: ["fb-cyan", "fb-orange", "fb-green"],
        iconImages: [
            "images/icons/prefer_cold.png",
            "images/icons/prefer_warmer.png",
            "images/icons/comfy.png",
        ],
    }
]

    • name: The internal name for the question.

    • type: Not yet implemented.

    • requiresAnswer: Array of object {name, value}. To depend on a previous questions answers, [{name: name, value: 11}]. The value "11" represents the first button on the left.

    • questionText: Title displayed at top of screen.

    • questionSecondText: Optional overflow title in case you need two lines.

    • iconText: Array of text to be displayed underneath icons, from left to right. Array length determines amount of options on screen.

    • iconColors: Array of fb-colours for the icons, from left to right. Array length determines amount of options on screen and should be the same length as the previous array.

    • iconImages: Array of png images for the icons, from left to right. Custom icons should have a square aspect ratio and be just below 1kb in size. Simply move them into resources/images/icons and point to them here. Array length determines amount of options on screen and should be the same length as the previous arrays.
